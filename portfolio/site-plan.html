Overview

Purpose

To showcase my programming projects and demonstrate my skills as a developer.

Audience

Potential employers, recruiters, and other developers interested in my projects.

Dynamic elements

I will have cards for each project I want to showcase. There will be functionality to emphasize the card when the user hovers the mouse over it, and when the user clicks on a card, it will show a dropdown with more information about the project and a link to the GitHub repository, if there is one. I will also only display the project title and a shortened description for mobile devices, and display an overview of the project when viewed on a desktop. I will also have a Contact page with a form that users can fill out to contact me. The form will have fields for the user's name, email, and message, and will have a submit button that will send the information to my email address.

The "About" button will jump to the About section of the page. As a stretch requirement, I will attempt to include a filter at the top of the page with label buttons that will filter the projects based on languages used. I will also have links to my LinkedIn and GitHub profiles.

Offtrack Storage CRM

This Refine project was generated with create refine-app. The CRM also uses open source code from Refine's Development Team. CRM Overview The OffTrack CRM is still in development, and will provide a centralized database for customer information, accessible through an intuitive and simple UI. Key features will include: Customer status filters (e.g., overdue payments, pending storage drop-offs) Contact management for scheduling storage pickups and drop-offs Billing and communication tools (e.g., email lists for invoicing) (Future potential feature) Route planning for more efficient customer service. This application will be used on both the computer and phone through the web. It will utilize Refine (a React-based framework) and Ant Design. While different CRM solutions exist, they may be too generic or expensive for a small business. This CRM is tailored to a business's specific needs and will provide an affordable alternative with flexibility for customization.

Artillery Simulator

This project is a simple artillery simulator built in C++. It allows users to input the angle and velocity of a projectile, and then simulates the trajectory of the projectile using basic physics equations. The simulator will display the path of the projectile and calculate the distance it travels. This project is a great way to demonstrate my understanding of physics and programming concepts, as well as my ability to create interactive applications.

View Project on GitHub

STL Implementation

These files are an implementation of the basic data structure classes and associated methods from the Standard Template Library in C++. Each data structure is implemented through test-driven development and has been completed using the pair-programming method. They are implemented to function exactly as they do in the STL.
